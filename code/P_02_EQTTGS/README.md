# EQT-Baryon: Event-based Baryon Structure in Energy Quantum Theory

This repository provides numerical support for the preprint:  
**"The Generative Structure of Baryons: An Event-Ontological Mechanism in Energy Quantum Theory"**

## Features
- Y-type baryon topology simulation
- Proton-neutron mass difference from frequency difference
- Baryon octet mass spectrum fitting
- Stability comparison: Star vs Triangle topology

## Requirements
- Python 3.8+
- numpy
- matplotlib
- scipy (for fitting)

## Installation
```bash
pip install numpy matplotlib scipy
```
## Usage
```bash
python stability_test.py        # Compare stability
python mass_difference.py       # Compute Î”m = 1.3 MeV
python baryon_spectrum_fit.py   # Fit baryon octet masses
```
